自分専用の英会話ジム。 solo-eikaiwa — a research-grounded English speaking gym for daily self-study

毎日5分から。録音・文字起こし・練習履歴はMacに保存し、AI処理のテキストだけを選択したproviderへ送信します。既定はClaude、完全ローカル構成も選べます。

デスクトップ版を入手 セットアップ手順

Local-first

保存はMac。AIの送信先は自分で選べる

ブラウザ録音 → whisper.cpp（ローカルSTT） → 選択したLLM（既定Claude） → 選択したTTS（キーなしはmacOS say）
  • 録音・文字起こし・練習履歴はMacに保存。マイク音声はローカルSTTで処理し、クラウドへ送信しません
  • 既定LLMはClaude。発話テキスト・プロンプト・必要な会話文脈を、用途ごとに選んだClaude / Codex / OpenAI公式 / OpenAI互換接続先へ送信します
  • 完全ローカルは選択可能な構成。LLMをloopbackのOllama等、TTSをmacOS sayやloopbackへ向ければ、音声もテキストもMac外へ出ません

Daily menu

毎日のメニュー

🔊

クイックドリル（5〜10分）

音読ウォームアップ・くり返しトーク（4/3/2）・ロールプレイ・シャドーイングから1日1本。「頻度と完了」が上達に効くという研究知見どおり、短くても毎日回せる設計。

🗣️

AI ロールプレイ & 自由会話

レストラン・会議・障害対応など、場面別シナリオで AI と役割練習。詰まったら日本語で「言い方ヒント」。相手は AI だから、間違えても恥ずかしくない。

📖

暗記例文390 + 間隔反復

会話でそのまま使える390文を「日本語を見て声に出す→答え合わせ→自己評価」の産出リトリーバルで。評価に応じた間隔反復（SRS）が次の出題日を決める。

🎧

多聴ライブラリ

レベルに合った2〜4分の英語をスクリプトを隠したまま通しで聴く。日本語訳と表現解説つき。ノルマなし。

📐

レベル連動の難易度

4/3/2 の持ち時間、モデルトークの語彙と構文、お題の帯域までレベルに自動追従。レベルは XP で自動で上がり、ステージ境界の昇格・降格だけは実績根拠つきの「提案 + あなたの承認」。XP は減らない。

📈

進捗と月次レビュー

調音速度・練習カレンダー（練習量で濃くなる草）・レベル履歴。30日ごとに練習データから日本語の振り返りレポートを生成。ノルマや判定はなし — 情報として見せるだけ。

Privacy

保存場所と送信先を分けて確認できる

構成 / providerMac外・別端末へ送るデータMac側の保存認証・課金先
Claude（既定）発話テキスト、プロンプト、必要な会話文脈録音・文字起こし・履歴・進捗Claude CLIのPro/Max、または選択時のみAnthropic API
Codex同じテキスト同上Codex CLIのChatGPTログイン、または選択時のみOpenAI API
OpenAI公式同じテキストをOpenAIのAPIへ送信同上OpenAI APIキー
OpenAI互換loopbackはMac外送信なし。LAN/remoteは画面に表示したoriginへ同じテキストを送信同上接続先の運営者。remoteではその認証・課金規約に従う
TTSクラウド利用時は読み上げるテキストのみ。マイク録音は送らない音声キャッシュ接続先。キーなし既定はmacOS sayで追加課金なし
完全ローカル構成外部送信なしすべてMac内クラウド認証・従量課金なし
  • 設定画面で場所とoriginを常時表示。OpenAI互換URLを、このMac（loopback）/ LAN / remote / 無効に分類します
  • remote切替は保存前に開示。プロンプトと文字起こしがMac外へ出ること、認証・データ取扱い・課金が接続先に従うことを表示します

Research-based

思いつきではなく、研究ベース

セッション構成は、第二言語習得（SLA）研究のメタ分析・原典を敵対的に検証したリサーチレポートに基づいています。

4/3/2 時間圧反復

同じ話を制限時間を縮めながら繰り返すと発話速度と流暢性が向上する。準備フェーズと明示的フィードバックの併用でさらに効果。

産出リトリーバル

「思い出して口に出す」は受容型の学習より記憶定着に有効。例文練習・聴覚起点モードの核。

頻度 > 総時間

1日サボっても無害、1週間空くと有害。だから「短くても毎日」を最優先する導線設計。連続日数を煽る演出は意図的に不採用。

動機づけ研究の知見に沿って、ノルマ・警告・喪失演出は一切ありません。フィードバックはすべて「情報」として提示され、判断は常にあなたに委ねられます。

Get started

はじめる

デスクトップアプリ（推奨）

Apple Silicon Macなら、GitHub Releases の dmg をダウンロードし、アプリケーションフォルダへコピーして起動するだけです。Bun や Homebrew は不要です。v0.29.0以降は、起動時の案内から更新を適用できます。

ソースから常駐

macOS + Bun + Homebrew を用意し、初回だけ ./scripts/setup.sh、続けて ./scripts/install-daemon.sh を実行します。以後の更新は再登録せず、READMEのビルド・再起動手順に従います。

既定の AI は Claude（Claude Pro/Max サブスクで動作）。Ollama / Codex / OpenAI公式 / OpenAI互換への切替は設定画面から行えます。
