# Changelog

このプロジェクトの特筆すべき変更を記録します。形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に、バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

## [0.2.0] - 2026-07-06

### Added

- **暗記例文300**: 会話用の完全オリジナル例文300文（文法・機能25分類×12文、日常/ビジネス/IT均等）。産出リトリーバル型の練習フロー（日本語→口頭→答え合わせ＋音声→3段階自己評価）と間隔反復（1/3/7/14/30/60日）、一覧モード、音声一括生成スクリプト
- **サイドバーレイアウト**: ホーム/自由会話/ライブラリ/暗記例文300のナビと練習記録ウィジェット
- **ホーム刷新**: 説明文付きドリルカード（4色アイコンタイル）、日替わり「今日の学習を始める」CTA、可変幅のGitHub風練習日カレンダー（週頭日付ラベル付き）
- **お題拡充**: 一般ビジネス・日常会話のお題10種を追加（計22種）
- **Tailwind CSS v4 + shadcn/ui 基盤**（テーマ変数でライト/ダーク対応）

### Changed

- 4/3/2 の準備チャンクを穴あきフレームから具体例入りの完全文に変更（音読可能に）
- 練習日カレンダーの列を1列=7日（曜日）に統一

### Removed

- if-then アンカー（「続けるコツ」）機能 — 研究上の効果量が小さく、UIの意図が不明瞭だったため

## [0.1.0] - 2026-07-06

最初のリリース。検証済みの第二言語習得（SLA）研究に基づく、ローカルファーストな英会話独習アプリ。

### Added

- **音声ループ**: ブラウザ録音 → whisper.cpp（ローカルSTT）→ Claude（会話相手・コーチ、Agent SDK 経由でサブスクリプション認証）→ OpenAI TTS（未設定時は macOS `say` にフォールバック）
- **クイックドリル（5〜10分・日次のデフォルト導線）**: 音読ウォームアップ / 4/3/2ミニ / ロールプレイ / シャドーイングの単品ドリル
- **強化セッション（60分 / 30分）**: 音読ウォームアップ → 4/3/2 流暢性トレーニング → 実務ロールプレイ → シャドーイング → 振り返り の通し構成
- **4/3/2 流暢性トレーニング**: 準備フェーズ（お題別の表現チャンク・話の骨組み・モデルトーク聴取）→ 時間圧ラウンド（2分→1.5分→1分、メニュー設定駆動）→ ラウンド間の明示的フィードバック（エラーログ＋日本語解説）
- **自由会話モード**（AIとの英語フリートーク）
- **コンテンツローテーション**: お題（`content/topics/`）とシナリオ（`content/scenarios/`）を frontmatter 付き Markdown で管理、least-recently-used で自動選択
- **チャンク音声再生**: 各表現の🔊ボタン（押下時のみ生成、以降はキャッシュ即再生）
- **モデルトーク**: 準備画面表示時の先読み生成・段階表示・本文の常時表示
- **ライブラリ**: 生成したモデルトークを SQLite（`bun:sqlite`）に記録し、一覧・本文確認・再再生
- **練習日カレンダー**（直近8週・情報表示のみ）と **if-then アンカー**（習慣化研究に基づく1行プラン）
- **ローカルHTTPS**: Caddy 用サイト定義（`https://learn-english`）
- **セッションログ**: 全イベントを JSONL に記録（将来のメトリクス分析用・ローカル専用）
- **ドキュメント**: 設計スペック、実装計画、ディープリサーチレポート3本（流暢性・語彙/チャンク・継続/習慣化 — 主要クレームは原典まで検証済み）

### Security / Privacy

- サーバは 127.0.0.1 バインドのみ（外部非公開）
- 学習データ（録音・トランスクリプト・進捗・DB・キャッシュ）は `data/` のローカルファイルで、リポジトリにコミットされない
- 音声データはマシン外に出ない（外部送信はテキストのみ: Claude への発話テキスト、OpenAI TTS への応答テキスト）
