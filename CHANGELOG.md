# Changelog

このプロジェクトの特筆すべき変更を記録します。形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に、バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

## [0.12.1] - 2026-07-07

### Fixed

- 多聴の段落間の無音を解消（再生中に次の段落の音声を先読み）
- 例文練習のトグル（ヒント非表示・音から始める）を、効果のない一覧タブでは非表示に
- 素材生成CLI: 一時的な通信エラーでも例文・お題生成が再試行するように（多聴と同じ挙動に統一）。多聴素材の検証を強化（本文長・タイトルの禁止文字）
- サポート設定 ⓘ ボタンの読み上げ対応を改善（スクリーンリーダー向け）

## [0.12.0] - 2026-07-07

### Added

- **例文練習の「音から始める」モード（P6-3）**: 練習ツールバーのトグルをオンにすると、次のカードから英文・日本語とも見せずに音声だけを先に再生。意味を思い浮かべて（または口に出して）から「答えを見る」で答え合わせ。自己評価・間隔反復はこれまでと同一。研究根拠: 聴覚形式の語彙知識と産出リコールがリスニング理解の最良予測子

## [0.11.0] - 2026-07-07

### Added

- **多聴ライブラリ（P6-2）**: サイドバーに「多聴」を新設。レベルに合った短い英語（3ドメイン×難易度2帯・各2〜4分）を段落ごとの音声で通し再生。スクリプトは隠して聞くのが既定（ボタンで表示・日本語訳と解説も可）。聴いた本数は「今週n本」の情報表示のみ（ノルマなし）。研究根拠: 多読・多聴はレベル適合の選書と記録があるときに効果が出る（d=0.50/0.47）
- 多聴素材の生成CLIモード（`scripts/generate-content.ts listening` — 語彙レベリング連動・検証付き一括書き込み）

## [0.10.0] - 2026-07-07

### Added

- **会話・生成の語彙レベリング（P6-1）**: レベル1〜30（ステージ1〜3）では、自由会話・ロールプレイ・モデルトーク・準備チャンク・例文/お題生成の英語を高頻度2,000〜3,000語族に制約（研究の95%カバレッジ基準）。上級ステージの挙動は完全に従来どおり
- **就寝前レビュー案内（P6-4）**: 20時以降にホームへ「寝る前の復習は定着に少し有利」の一言（情報表示のみ）

### Changed

- **サポート設定を個別トグル3つに簡素化**: 一括プリセット（自動/多め/少なめ）を廃止し、日本語ヒント・モデルトーク自動再生・歯抜け既定をそれぞれ 自動/オン/オフ で設定する方式に。設定済みの個別値はそのまま引き継がれます
- シャドーイングのスクリプトは常に「隠して開始」に（表示ボタンはこれまで通り）

## [0.9.0] - 2026-07-07

### Added

- **解説機能の全画面展開（P5）**: ライブラリに日本語訳＋解説ボタン、マイチャンクに「もっと詳しく」、レベル測定のお題に日本語訳トグル。例文練習の解説も再試行・言語切り替えに対応
- **サポート設定の ⓘ ヒント**: サイドバーの各設定（サポート/日本語ヒント/モデルトーク/歯抜け既定）に、自動・オン・オフが何をするかの説明ポップオーバー
- 月次レビューが生成済みのとき「今月は生成済み」と表示（従来は無反応）

### Changed

- サイドバーの並びを整理: サポート設定 → 文字サイズ → 言語 → 練習実績
- セッション画面の外枠文言（戻るボタン・進捗ドット等）と 4/3/2 の副題表示を言語切り替えに対応

## [0.8.0] - 2026-07-07

### Added

- **練習系画面の英語UI対応（P4）**: 音読ウォームアップ・4/3/2・振り返り・ロールプレイ・自由会話・シャドーイング・ライブラリ・チャンク一覧の全文言を EN/日本語 切り替えに対応。サーバ生成のメニュータイトルも言語別に表示（日本語表示は従来と完全に同一）

### Changed

- クライアント内部を再編（R4・外部挙動は不変）: API層をドメイン別モジュールへ分割、i18n を画面別サブ辞書の合成に再構成、例文練習画面をタブ別ファイルへ分割、行の音声再生・解説ボタン・サポート設定購読を共有フックに統一
- シャドーイングの訳解説が失敗したとき再試行ボタンを表示（従来は再試行不能）

## [0.7.0] - 2026-07-07

### Added

- **学習サポート設定（P2）**: サイドバーに常設の「サポート」パネル。自動（レベル連動）/多め/少なめのプリセットと、日本語ヒント・モデルトーク・歯抜け既定の個別トグル。設定はブラウザに保存され、開いている画面にも即反映
- **段階的ステップの横展開（P3）**: シャドーイングは「スクリプトを隠して聞く」が既定に（ボタンでいつでも表示・サポート多めなら最初から表示）。音読ウォームアップに任意の「歯抜けで音読」2周目。振り返り・4/3/2フィードバックの訂正に「もっと詳しく」ボタン（なぜその言い方が自然かの日本語解説）

### Changed

- チャンクの日本語訳は常にデータとして返し、見せるかどうかだけを画面側で制御するように（上級ステージでも設定やボタンでいつでも日本語を確認できる）
- モデルトークの「なし」を廃止し、どのステージでも聞く手段（ボタン）を残すように

## [0.6.0] - 2026-07-07

### Added

- **会話中の日本語支援（P1）**: 自由会話・ロールプレイの各 AI 発話に「訳」ボタン（訳のみ・本文ハッシュでキャッシュ・失敗時は再試行ボタン）。詰まったときに言いたいことを日本語で入力すると英語の言い方を2〜3個提案する「言い方ヒント」（直近の会話6ターンを文脈として利用・明示操作でのみ表示）
- **ロールプレイの開始フレーズ**: 全16シナリオに会話の切り出しフレーズ2〜3個を追加し、開始前の準備カードに表示

### Changed

- サーバHTTP層を機能別ルータモジュール（13ドメイン）＋テーブルディスパッチに再編（R1・外部挙動は不変）。テストも型付き共有フェイクでドメイン別に分割
- クライアントの非同期ロードを `useLoad` フックに統一（R2）。開発時の振り返り画面の二重POSTを解消

## [0.5.0] - 2026-07-06

### Added

- **月次レビュー（M5）**: Progress 画面で直近30日の練習データ（練習時間・調音速度・例文の定着・苦手カテゴリ・収集チャンク・レベル遷移）をまとめた日本語の振り返りレポートを生成。同一月は再生成せずキャッシュを返す（情報表示のみ・ノルマや判定なし）。過去レポートの一覧・展開も可能
- **実力データ駆動のコンテンツ生成CLI（M5）**: `scripts/generate-content.ts` — SRS の bad 率が高いワースト3カテゴリに新規例文を4文ずつ追記する `sentences`、現在レベル向けのお題2本＋シナリオ1本を生成する `topics`。書き込み前バリデーション（不正なら書かず非ゼロ終了）・全件検証後の一括書き込み・`--dry` プレビュー対応
- **ロールプレイの3種分割**: クイックドリルが日常・ビジネス・IT の3枚に。タイトルも実際のシナリオのドメイン表記に
- **シャドーイングの日本語訳＋表現解説ボタン**（本文ハッシュでキャッシュ）
- **レベル測定の常設導線**: サイドバー「ホーム」直下に Level Check を常設。導線カードは未測定時ヒーロー直下・月次時カレンダー手前の専用スタイルに
- UX一貫性レビューと次期計画（P1〜P5）を docs に追加

### Changed

- 日替わり「今日の学習を始める」CTA を削除（カード1クリックと重複のため）
- README のロードマップを次期計画（P1〜P5）に更新

## [0.4.0] - 2026-07-06

### Added

- **詰まった表現の自動チャンク収集（M3）**: 4/3/2 のフィードバックと振り返りの修正案から「あなたの発話→正しい言い方」のペアを自動収集（1日5件まで・重複除去）。翌日から暗記例文と同じ SRS キューの先頭に混ざり、「あなたの表現を正しく言い直す」カードで復習。一覧の My chunks セクションから削除も可能
- **スピーキングメトリクスと進捗ダッシュボード（M4）**: whisper のセグメントタイムスタンプから調音速度（ポーズ除外WPM）・ポーズ回数/時間・繰り返し率を発話ごとに記録。サイドバーの Progress 画面に練習時間・調音速度トレンド・ポーズ/繰り返し・レベル履歴を表示（情報表示のみ・目標演出なし）
- **例文練習の学習ステップ強化**: 「歯抜けを見る」段階表示（決定的cloze・同じ文は毎回同じ穴）、ヒント非表示設定（答え表示時は常にヒント表示）、「もっと詳しく」解説
- **解説300件を同梱**: 全例文の詳しい解説（組み立ての考え方・使い回し例・よくある間違い）を事前生成してリポジトリに同梱。キーなしで即表示
- **UI言語切り替え**: サイドバーの EN/日本語トグル（デフォルト英語）。ホーム・例文画面・ダッシュボード等の文言が対象
- ファビコン（緑スクエア＋吹き出し）

### Changed

- サーバの日付処理をローカル日付に統一（練習日カレンダー・セッションログ・メニューキャッシュの深夜0時ズレを解消）
- README を利用者向け（できること主導）に全面改稿

## [0.3.0] - 2026-07-06

### Added

- **レベル・XP システム**: レベル（Lv1〜・上限なし）が 4/3/2 の持ち時間、準備チャンク数、ヒント言語、モデルトーク表示、お題の帯域を駆動。ブロック完了・例文自己評価・測定完了で XP が貯まり、ステージ内は自動レベルアップ、ステージ境界（Lv10/20/30/40/50）は実績根拠つきの提案＋承認制。降格は中立表現の提案のみ（自動降格なし・XP は減らない）
- **プレースメント（レベル測定）**: 自己紹介→状況説明→意見の3タスク（約10分）を録音し、CEFR 記述子ベースのルーブリックで開始レベルを提示。反映は利用者の確定操作制。30日ごとに月次測定の導線をホームに表示
- **コンテンツの3ドメイン化**: お題・シナリオに `domain`（daily/business/it）と `level`（適合ステージ範囲）タグを追加し、選択はドメインのラウンドロビン＋帯域フィルタ＋LRU。日常会話4本・一般ビジネス4本のロールプレイシナリオを新規追加（計16本）
- サイドバーにレベルゲージ（クリックで手動変更）、ホームに昇格/降格提案カード

### Changed

- 4/3/2 のラウンド秒数が固定値（120/90/60）からレベル連動（90〜180秒の線形）に。デフォルト開始は Lv13（従来よりやや易しい）
- 日次メニューは日内固定になり、明示的なレベル変更（提案の承認・手動設定・測定の反映）時のみ当日再構築

## [0.2.0] - 2026-07-06

### Added

- **暗記例文300**: 会話用の完全オリジナル例文300文（文法・機能25分類×12文、日常/ビジネス/IT均等）。産出リトリーバル型の練習フロー（日本語→口頭→答え合わせ＋音声→3段階自己評価）と間隔反復（1/3/7/14/30/60日）、一覧モード、音声一括生成スクリプト
- **サイドバーレイアウト**: ホーム/自由会話/ライブラリ/暗記例文300のナビと練習記録ウィジェット
- **ホーム刷新**: 説明文付きドリルカード（4色アイコンタイル）、日替わり「今日の学習を始める」CTA、可変幅のGitHub風練習日カレンダー（週頭日付ラベル付き）
- **お題拡充**: 一般ビジネス・日常会話のお題10種を追加（計22種）
- **Tailwind CSS v4 + shadcn/ui 基盤**（テーマ変数でライト/ダーク対応）

### Changed

- 4/3/2 の準備チャンクを穴あきフレームから具体例入りの完全文に変更（音読可能に）
- 練習日カレンダーの列を1列=7日（曜日）に統一

### Removed

- if-then アンカー（「続けるコツ」）機能 — 研究上の効果量が小さく、UIの意図が不明瞭だったため

## [0.1.0] - 2026-07-06

最初のリリース。検証済みの第二言語習得（SLA）研究に基づく、ローカルファーストな英会話独習アプリ。

### Added

- **音声ループ**: ブラウザ録音 → whisper.cpp（ローカルSTT）→ Claude（会話相手・コーチ、Agent SDK 経由でサブスクリプション認証）→ OpenAI TTS（未設定時は macOS `say` にフォールバック）
- **クイックドリル（5〜10分・日次のデフォルト導線）**: 音読ウォームアップ / 4/3/2ミニ / ロールプレイ / シャドーイングの単品ドリル
- **強化セッション（60分 / 30分）**: 音読ウォームアップ → 4/3/2 流暢性トレーニング → 実務ロールプレイ → シャドーイング → 振り返り の通し構成
- **4/3/2 流暢性トレーニング**: 準備フェーズ（お題別の表現チャンク・話の骨組み・モデルトーク聴取）→ 時間圧ラウンド（2分→1.5分→1分、メニュー設定駆動）→ ラウンド間の明示的フィードバック（エラーログ＋日本語解説）
- **自由会話モード**（AIとの英語フリートーク）
- **コンテンツローテーション**: お題（`content/topics/`）とシナリオ（`content/scenarios/`）を frontmatter 付き Markdown で管理、least-recently-used で自動選択
- **チャンク音声再生**: 各表現の🔊ボタン（押下時のみ生成、以降はキャッシュ即再生）
- **モデルトーク**: 準備画面表示時の先読み生成・段階表示・本文の常時表示
- **ライブラリ**: 生成したモデルトークを SQLite（`bun:sqlite`）に記録し、一覧・本文確認・再再生
- **練習日カレンダー**（直近8週・情報表示のみ）と **if-then アンカー**（習慣化研究に基づく1行プラン）
- **ローカルHTTPS**: Caddy 用サイト定義（`https://learn-english`）
- **セッションログ**: 全イベントを JSONL に記録（将来のメトリクス分析用・ローカル専用）
- **ドキュメント**: 設計スペック、実装計画、ディープリサーチレポート3本（流暢性・語彙/チャンク・継続/習慣化 — 主要クレームは原典まで検証済み）

### Security / Privacy

- サーバは 127.0.0.1 バインドのみ（外部非公開）
- 学習データ（録音・トランスクリプト・進捗・DB・キャッシュ）は `data/` のローカルファイルで、リポジトリにコミットされない
- 音声データはマシン外に出ない（外部送信はテキストのみ: Claude への発話テキスト、OpenAI TTS への応答テキスト）
