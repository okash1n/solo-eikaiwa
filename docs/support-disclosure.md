# 支援情報の開示契約

支援情報は想起や聞き取りの前に答えを見せないため、設定だけでは表示しません。設定は表示ボタンの利用可否や事前準備を決めるだけで、本文・訳・解説・スクリプトは各画面での操作後に開示します。

| 画面 | 支援情報 | 初期状態 | 開示操作 | リセット境界 |
| --- | --- | --- | --- | --- |
| 音読ウォームアップ | 準備チャンク・代替表示のお題ヒントの日本語訳 | 非表示 | 日本語ヒントを表示 | お題 |
| 4/3/2 準備・ラウンド | 準備チャンク・お題の骨組みの日本語訳 | 非表示 | 日本語ヒントを表示 | お題 / セッション |
| 4/3/2 準備 | モデルトークの音声・スクリプト | 非表示 | モデルトークを聞く、続けてスクリプトを開く | セッション |
| 例文練習 | 答え・詳しい解説 | 非表示 | 答えを見る / もっと詳しく | 次のカード |
| 多聴 | スクリプト・日本語訳と解説 | 非表示 | スクリプトを表示 / 日本語訳と解説 | 素材 |
| シャドーイング | スクリプト・日本語訳と解説 | 非表示 | スクリプトを表示 / 日本語訳と解説 | 素材 |
| モデルトークライブラリ | スクリプト・日本語訳と解説 | 非表示 | スクリプトを開く / 日本語訳と解説 | エントリ |
| 自由会話・ロールプレイ | AI発話の訳 | 非表示 | 訳 | 発話ターン |
| 自由会話・クイックロールプレイ（終了後） | 遅延訂正の振り返り（訂正最大3件・詳しい解説） | 非表示 | 会話を振り返る（任意）→ もっと詳しく | 練習（会話の終了操作ごと） |
| 振り返り | 表現の詳しい解説 | 非表示 | もっと詳しく | 修正項目 |
| マイフレーズ一覧 | 表現の詳しい解説 | 非表示 | もっと詳しく | エントリ |

暗記例文・マイフレーズの一覧・検索タブは、学習済みの内容を見返すための参照ビューであり、英文と日本語訳を常時並べて表示します。想起前の答えを隠すこの開示契約の対象外です（ただし上記のとおり、マイフレーズの「もっと詳しく」解説だけは操作後に開示します）。

学習者自身の文字起こしは、録音停止・送信という明示操作の結果として表示する本人の記録であり、答えや訳などの支援情報とは区別します。
